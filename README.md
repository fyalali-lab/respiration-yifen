# VitalCoach — HRV Biofeedback Coaching

Application web de coaching biofeedback connectée au Polar H10 via Web Bluetooth.

## Démarrage rapide

```bash
npm install
npm run dev
```

Ouvrir **http://localhost:5173** dans **Chrome** ou **Edge**.

## Déploiement Vercel

```bash
# Via CLI
npm i -g vercel
vercel

# Ou connecter le repo GitHub à vercel.com
```

## Prérequis

- Chrome ou Edge (Web Bluetooth requis)
- Polar H10 (ou toute ceinture BLE Heart Rate compatible)
- HTTPS obligatoire en production (Vercel le gère automatiquement)
